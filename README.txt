READ ME BEFORE COMPILING
---------------------------------------------------
1. Data file has to be in the same folder as the java files
2. Entering the wrong file name or file not in the same folder will result in termation of program

COMPILING INSTRUCTIONS
----------------------------------------------------
1. To Compile, open command prompt at the file location and type in "javac QueueSim.java"
2. Once compiled, to execute, type in "java QueueSim <filename>"
